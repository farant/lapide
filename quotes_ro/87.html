Ispita este de mai multe feluri: una este importună, care stăruie cu nerușinare; alta este îndoielnică, care învăluie mintea într-un nor de nesiguranță; a treia este bruscă, care anticipează judecata rațiunii; a patra este ascunsă, care scapă ordinii deliberării; a cincea este violentă, care depășește puterile noastre; a șasea este frauduloasă, care seduce mintea; a șaptea este încurcată, care este împiedicată de căi felurite.

— Sf. Bernard, in Cornelius a Lapide, Comentariu la Facerea, Capitolul III
