O obediente fiel não conhece demoras, foge do amanhã; ignora a tardança, antecipa-se a quem manda; prepara os olhos para ver, os ouvidos para ouvir, a língua para falar, as mãos para a obra, os pés para o caminho; recolhe-se inteiramente para cumprir a vontade de quem ordena.

— São Bernardo, in Cornelius a Lapide, Comentário sobre o Génesis, Capítulo XVII — «Imediatamente naquele dia»
