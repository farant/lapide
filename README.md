# lapide


Hi! This repo is full of translations of Cornelius a Lapide's commentary on the Bible (plus some other cool texts).

* I have been doing all the translations with Claude Opus 4.6
* I have used these PDFs to do the translation: [http://cdigital.dgb.uanl.mx/la/1080014741_C/1080014741_C.html](http://cdigital.dgb.uanl.mx/la/1080014741_C/1080014741_C.html)
* I am using [Surya](https://github.com/datalab-to/surya) to do OCR on my computer.
* Claude then uses the OCR output (which is in Latin) to create an English translation.
* THEN Claude builds HTML pages based on the English translation.
* After this I create Latin HTML pages and try to match the structure of the English HTML.
* When I translate into other languages I try to make sure Claude uses English and Latin as reference.
* The PDFs have footnotes that were added by later authors and I am trying to leave these out so the texts are focused on Lapide's work.

All the translations, HTML, etc. in this repo are [CC0 1.0](https://creativecommons.org/publicdomain/zero/1.0/) and in the public domain.
