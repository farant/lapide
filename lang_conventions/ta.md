# Tamil (தமிழ்)

Use formal literary Tamil (செந்தமிழ் / எழுத்துத் தமிழ்). File suffix `_ta`. Tamil script (தமிழ் எழுத்து), LTR (no `dir` attribute needed).

## Biblical proper names

Follow Tamil Catholic Bible (திருவிவிலியம்) tradition: மோசே (Moses), ஆபிரகாம் (Abraham), ஈசாக்கு (Isaac), யாக்கோபு (Jacob), ஆரோன் (Aaron), தாவீது (David), யோசுவா (Joshua), எசாயா (Isaiah), எரேமியா (Jeremiah), நோவா (Noah), ஆதாம் (Adam), ஏவாள் (Eve), இயேசு (Jesus), பவுலு (Paul), பேதுரு (Peter), யோவான் (John).

## Saint names

Use புனித (Punitha) prefix — the standard Catholic Tamil convention. **Church Father names use Latinized forms** (not anglicized), preserving the Latin scholarly texture of Lapide's original.

Key mappings: புனித அகுஸ்தீனுஸ் (Augustine), புனித ஹியரோனிமுஸ் (Jerome), புனித பசிலியுஸ் (Basil), புனித அம்புரோசியுஸ் (Ambrose), புனித கிறிசோஸ்தோமுஸ் (Chrysostom), புனித கிரகோரியுஸ் (Gregory), புனித பெர்னார்துஸ் (Bernard), புனித ஹிலாரியுஸ் (Hilary), புனித சிப்ரியானுஸ் (Cyprian), புனித யுஸ்தீனுஸ் (Justin), புனித கிளமெண்ட்ஸ் (Clement), புனித அத்தனாசியுஸ் (Athanasius), புனித எப்பிபானியுஸ் (Epiphanius), புனித இஞ்ஞாசியுஸ் (Ignatius), தமஸ்கஸின் யோவான் (John of Damascus), புனித போனவெந்தூரா (Bonaventure), புனித பிரான்சிஸ்குஸ் (Francis), புனித தோமினிக்குஸ் (Dominic).

Names already close to Latin are kept as-is: தோமாஸ், லியோ, எப்ரேம்.

## Quotation marks

Use standard double quotation marks "..." for quotations.

## Bible references and psalm numbering

The Tamil Catholic Bible (திருவிவிலியம்) follows Vulgate psalm numbering. Keep Vulgate numbers as in the source text.

Gospel references use standard Tamil Bible name யோவான் (e.g., "யோவான் 8:12").

## Register

Tamil script has unique vowel-consonant combinations (உயிர்மெய்யெழுத்துக்கள்) that must be correctly formed. Proper use of pulli (்), kompu (ோ, ோ), and all vowel signs is critical. Tamil also uses Grantha consonants (ஜ, ஷ, ஸ, ஹ) for Sanskrit/foreign loanwords — these must be used correctly for proper names.

Use Arabic/Western numerals (1, 2, 3) rather than Tamil numerals (௧, ௨, ௩), matching modern Tamil Catholic Bible conventions.

## Agent pitfalls

- **Script confusion**: Tamil script is visually distinct from Devanagari (Hindi) and other Indic scripts but agents may still mix characters. Every character must be from the Tamil Unicode block (U+0B80–U+0BFF) plus Grantha consonants where needed.
- **Anglicized Church Father names**: Agents default to anglicized forms instead of the required Latinized forms. Must be explicitly instructed.
- **Colloquial register**: Agents may drift to spoken Tamil (பேச்சுத் தமிழ்) instead of literary Tamil (எழுத்துத் தமிழ்). Watch for colloquial verb forms and shortened pronouns.
- **Missing pulli**: Agents may omit the pulli (்) on consonants, changing meaning (e.g., க vs க்). This is the most common script error.
- **Grantha consonant avoidance**: Agents may avoid Grantha letters (ஜ, ஷ, ஸ, ஹ) and substitute native Tamil consonants, producing incorrect transliterations of proper names.
- **Verb agreement**: Tamil has strict subject-verb agreement for person, number, and gender. Agents may default to masculine singular forms.
