# Indonesian

Use bahasa baku (formal/standard Indonesian). File suffix `_id`. Latin alphabet, LTR (no `dir` attribute needed).

## Biblical proper names

Follow Indonesian convention: Musa (Moses), Yosua (Joshua), Yesaya (Isaiah), Yeremia (Jeremiah), Yakub (Jacob), Ibrahim (Abraham), Ishak (Isaac), Harun (Aaron), Daud (David).

## Saint names

Use Santo/Santa prefix: Santo Hieronimus (St. Jerome), Santo Basilius (St. Basil), Santo Agustinus (St. Augustine).

## Quotation marks

Use standard double quotation marks "..." for quotations.

## Register

Indonesian has no diacritics concern (standard Latin alphabet without accents).
