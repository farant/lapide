# Indonesian

Use bahasa baku (formal/standard Indonesian). Biblical proper names follow Indonesian convention: Musa (Moses), Yosua (Joshua), Yesaya (Isaiah), Yeremia (Jeremiah), Yakub (Jacob), Ibrahim (Abraham), Ishak (Isaac), Harun (Aaron), Daud (David). Saint names use Santo/Santa (e.g., Santo Hieronimus, Santo Basilius, Santo Agustinus). Use standard double quotation marks "..." for quotations. Indonesian has no diacritics concern (standard Latin alphabet without accents).
