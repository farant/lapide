# Portuguese

Use European Portuguese (traditional orthography: facto, exactamente, correctamente — not Brazilian fato, exatamente, corretamente). European Portuguese is appropriate for the formal scholarly/theological register of the source material and its European ecclesiastical context. The text remains fully accessible to Brazilian readers, as differences between the two standards are minimal in formal theological writing. File suffix `_pt`. Latin alphabet, LTR (no `dir` attribute needed).

## Biblical proper names

Follow Portuguese Bible tradition (Bíblia Sagrada, Difusora Bíblica / Almeida): Adão (Adam), Eva (Eve), Noé (Noah), Abraão (Abraham), Isaac (Isaac), Jacob (Jacob), Esaú (Esau), José (Joseph), Moisés (Moses), Josué (Joshua), Isaías (Isaiah), Jeremias (Jeremiah), David (David), Salomão (Solomon), Caim (Cain), Abel (Abel), Set (Seth), Henoque (Enoch), Matusalém (Methuselah), Lameque (Lamech), Enós (Enos/Enosh), Cainã (Cainan), Malaleel (Mahalalel), Jared (Jared), Sem (Shem), Cam (Ham), Jafé (Japheth), Elias (Elijah), Eliseu (Elisha), Aarão (Aaron), Gedeão (Gideon), Sansão (Samson), Débora (Deborah), Rute (Ruth), Samuel (Samuel), Ezequias (Hezekiah), Jonas (Jonah), Daniel (Daniel), Ezequiel (Ezekiel), Malaquias (Malachi).

## Saint names

Use "São" before names beginning with a pronounced consonant (São Jerónimo, São Basílio, São Crisóstomo, São Bernardo, São Leão, São Paulo, São João, São Pedro, São Tomás, São Mateus, São Lucas, São Marcos, São Gregório, São Cipriano, São Clemente, São Justino, São Boaventura, São Francisco, São Domingos) and "Santo" before names beginning with a vowel sound (Santo Agostinho, Santo Ambrósio, Santo Atanásio, Santo Efrém, Santo Inácio, Santo Ireneu, Santo Epifânio, Santo Anselmo, Santo Isidoro, Santo Hilário). Since H is silent in Portuguese, H-initial names are treated as vowel-initial (Santo Hilário, not São Hilário). Feminine form: Santa (Santa Marcela). "São/Santo/Santa" is capitalized in Portuguese (unlike French "saint").

"Beato" for beatified but not canonized figures (e.g., "O Beato Cirilo").

Key saint name forms: São Jerónimo (St. Jerome), Santo Agostinho (St. Augustine), São Basílio (St. Basil), Santo Ambrósio (St. Ambrose), São João Crisóstomo (St. John Chrysostom), São Gregório (St. Gregory), São Bernardo (St. Bernard), Santo Hilário (St. Hilary), São Cipriano (St. Cyprian), São Justino (St. Justin), São Clemente (St. Clement), Santo Atanásio (St. Athanasius), Santo Epifânio (St. Epiphanius), Santo Inácio (St. Ignatius), São João Damasceno (St. John of Damascus), São Boaventura (St. Bonaventure), São Francisco (St. Francis), São Domingos (St. Dominic), São Tomás (St. Thomas — referring to Aquinas), São Leão (St. Leo), São Paulo (St. Paul), Santo Efrém (St. Ephrem), São Gregório Nazianzeno (St. Gregory Nazianzen), Santo Ireneu (St. Irenaeus), Santo Isidoro (St. Isidore).

**Abbreviation:** The preferred form is the full "São/Santo/Santa". An abbreviated form "S." appears in some existing files (e.g., "S. Jerónimo", "S. Agostinho", "S. Vítor") but the full form is dominant and should be used consistently in new translations.

## Quotation marks

Use Portuguese guillemets «...» (without inner spaces, unlike French) for quotations. This is the standard Portuguese typographic convention. Example: «No princípio criou Deus o céu e a terra.» Scripture quotations within `<em>` tags also use guillemets: <em>«Na Vossa luz veremos a luz.»</em>

No nested quote convention has been established in the existing files — nested guillemets have not been observed. If needed, use double quotation marks "..." for inner quotes.

## Bible references and psalm numbering

Bible references use both comma and colon as verse separators in existing files (e.g., "Gén. 25,26" and "Génesis 2:1-25", "Salmo 36,10" and "Salmo 20:2"). **The preferred convention is comma** (e.g., "Génesis 1,1", "Salmo 36,10", "Êxodo 20,8"), which follows standard Portuguese Bible citation style. New translations should use the comma consistently.

Book names may be abbreviated (Gén., Êx., Col., Jo., Mt., Lc., Mc., Dt., Lv., Nm.) or written out in full (Génesis, Êxodo, Mateus, João, etc.). Either is acceptable; be consistent within a file.

Psalm numbering follows the Vulgate convention used in the source text. Roman numerals are sometimes used for psalm numbers (e.g., "Salmo LXXXIX, 4") but Arabic numerals are more common (e.g., "Salmo 36,10") and preferred for consistency.

## Register

European Portuguese, formal scholarly/theological register. Traditional orthography is used throughout:

- **-cção / -cção forms**: acção, selecção, colecção, protecção, direcção, distinção, operação (not ação, seleção, coleção, etc.)
- **-ct- clusters**: facto, exactamente, correctamente, correcta (not fato, exatamente, corretamente)
- **Spelling**: baptismo (not batismo), óptimo (not ótimo), adopção (not adoção)
- **Diacritics**: Proper use of Portuguese diacritics is critical: á, à, â, ã, é, ê, í, ó, ô, õ, ú, ç. Note "Génesis" (with accent), "Êxodo" (with circumflex), "Jerónimo" (with accent on the o), "Deuteronómio" (with accent).

The register employs formal constructions such as mesoclisis (e.g., "seguir-se-á"), future subjunctive, and literary vocabulary appropriate to theological commentary.

## Agent pitfalls

- **Brazilian Portuguese forms**: Agents frequently produce Brazilian Portuguese instead of European Portuguese. Watch for: "fato" (should be "facto"), "batismo" (should be "baptismo"), "ação" (should be "acção"), "exatamente" (should be "exactamente"), "ótimo" (should be "óptimo"), "adoção" (should be "adopção"). This is the most common and most important error to check.
- **Missing diacritics on book names**: Agents drop accents on biblical book names — "Genesis" instead of "Génesis", "Exodo" instead of "Êxodo", "Deuteronomio" instead of "Deuteronómio". These must be accented.
- **Abbreviated "S." instead of "São/Santo"**: Some existing files inconsistently use "S. Jerónimo" or "S. Agostinho". The full form ("São Jerónimo", "Santo Agostinho") is preferred and should be used in new translations.
- **Inconsistent verse separators**: Agents may mix colons and commas in Bible references (e.g., "Génesis 2:1" vs "Génesis 2,1"). Use comma consistently as the verse separator.
- **English-style quotation marks**: Agents default to English "..." quotation marks instead of guillemets «...». Must be explicitly instructed to use guillemets.
- **Spaces inside guillemets**: Unlike French, Portuguese guillemets do NOT have inner spaces. Write «texto», not « texto ».
- **Wrong saint title**: Using "São" before vowel-initial names ("São Agostinho") or "Santo" before consonant-initial names ("Santo Jerónimo"). The rule is: "Santo" before vowel sounds (including silent-H names like Santo Hilário), "São" before pronounced consonants.
- **Incorrect proper name forms**: Agents may use English or Latin forms instead of Portuguese — e.g., "Moses" instead of "Moisés", "Abraham" instead of "Abraão", "Noah" instead of "Noé", "Cain" instead of "Caim", "Seth" instead of "Set".
- **Hyper-correction of -ct- clusters**: Agents may over-apply the European Portuguese -ct- rule and produce non-existent forms like "perfectamente" (should be "perfeitamente"), "perfecto" (should be "perfeito"), or "perfecta" (should be "perfeita"). The -ct- cluster is only preserved in words where European Portuguese tradition retains it (facto, exacto, correcto, baptismo, óptimo, acção, etc.). The word "perfeito/perfeitamente" never had a -ct- cluster in Portuguese — it is always "perfeito", never "perfecto".
