# Russian

Use formal scholarly Russian (книжный/научный стиль). File suffix `_ru`. Cyrillic script, LTR (no `dir` attribute needed). `lang="ru"` on the `<html>` tag. Biblical proper names follow Russian Synodal Bible tradition (Синодальный перевод): Моисей (Moses), Авраам (Abraham), Иаков (Jacob), Исаак (Isaac), Аарон (Aaron), Давид (David), Иисус Навин (Joshua), Исаия (Isaiah), Иеремия (Jeremiah), Адам (Adam), Ева (Eve), Ной (Noah). Saint names use св. (святой/святая) prefix: св. Иероним (St. Jerome), свт. Василий Великий (St. Basil), блж. Августин (St. Augustine), свт. Амвросий Медиоланский (St. Ambrose), свт. Иоанн Златоуст (Chrysostom), свт. Григорий Великий (Gregory), св. Бернард Клервоский (Bernard), свт. Иларий Пиктавийский (Hilary), сщмч. Киприан Карфагенский (Cyprian), мч. Иустин Философ (Justin), сщмч. Климент Римский (Clement), свт. Афанасий Великий (Athanasius), свт. Епифаний Кипрский (Epiphanius), сщмч. Игнатий Богоносец (Ignatius), прп. Иоанн Дамаскин (John of Damascus), св. Бонавентура (Bonaventure), св. Франциск Ассизский (Francis), св. Доминик (Dominic), св. Фома Аквинский (Thomas), св. Лев Великий (Leo), св. Павел (Paul), прп. Ефрем Сирин (Ephrem). Use Russian quotation marks «...» for standard quotations and „..." for nested quotes. Russian Catholic/theological terminology: Священное Писание (Holy Scripture), таинство (sacrament), благодать (grace), грех (sin), спасение (salvation), Отец Церкви (Church Father), Евангелие (Gospel), Бытие (Genesis), Исход (Exodus). Gospel references use Russian Bible convention (e.g., «Ин. 8:12»). The register should be formal scholarly Russian suitable for theological commentary — книжный стиль, avoiding colloquial constructions. Note on saint prefixes: Russian theological tradition uses specific abbreviated prefixes indicating the type of sanctity (свт. = святитель/bishop-saint, блж. = блаженный/blessed, прп. = преподобный/monastic saint, сщмч. = священномученик/hieromartyr, мч. = мученик/martyr). These are standard in Russian ecclesiastical writing and should be used consistently. Proper use of Russian letters (ё where required, й vs и) is important. Agents must be explicitly instructed to use «...» quotation marks (not English "...") as they default to English-style quotes.
