# Russian

Use formal scholarly Russian (книжный/научный стиль). File suffix `_ru`. Cyrillic script, LTR (no `dir` attribute needed). `lang="ru"` on the `<html>` tag.

## Biblical proper names

Follow Russian Synodal Bible tradition (Синодальный перевод): Моисей (Moses), Авраам (Abraham), Иаков (Jacob), Исаак (Isaac), Аарон (Aaron), Давид (David), Иисус Навин (Joshua), Исаия (Isaiah), Иеремия (Jeremiah), Адам (Adam), Ева (Eve), Ной (Noah).

## Saint names

Use св. (святой/святая) prefix, with specific abbreviated prefixes indicating the type of sanctity per Russian theological tradition:

- свт. = святитель (bishop-saint)
- блж. = блаженный (blessed)
- прп. = преподобный (monastic saint)
- сщмч. = священномученик (hieromartyr)
- мч. = мученик (martyr)

These are standard in Russian ecclesiastical writing and should be used consistently.

## Quotation marks

Use Russian quotation marks «...» for standard quotations and „..." for nested quotes.

## Theological terminology

Священное Писание (Holy Scripture), таинство (sacrament), благодать (grace), грех (sin), спасение (salvation), Отец Церкви (Church Father), Евангелие (Gospel), Бытие (Genesis), Исход (Exodus).

## Bible references and psalm numbering

Gospel references use Russian Bible convention (e.g., «Ин. 8:12»).

## Register

Formal scholarly Russian suitable for theological commentary — книжный стиль, avoiding colloquial constructions. Proper use of Russian letters (ё where required, й vs и) is important.

## Agent pitfalls

- **English-style quotes**: Agents must be explicitly instructed to use «...» quotation marks (not English "...") as they default to English-style quotes.
- **Saint prefix types**: Agents may use generic св. for all saints instead of the specific prefix (свт., блж., прп., сщмч., мч.) appropriate to each saint's type of sanctity.
- **ё omission**: Agents may use е where ё is required.
