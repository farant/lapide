# Swedish

Use formal scholarly Swedish (formellt akademiskt språk). File suffix `_sv`. Latin alphabet, LTR (no `dir` attribute needed). `lang="sv"` on the `<html>` tag. Biblical proper names follow Swedish Bible tradition (Bibel 2000 / Svenska Folkbibeln): Mose (Moses), Abraham, Isak (Isaac), Jakob (Jacob), Aron (Aaron), David, Josua (Joshua), Jesaja (Isaiah), Jeremia (Jeremiah), Adam, Eva (Eve), Noa (Noah). Saint names use S:t/S:ta (Sankt/Sankta) prefix following Swedish Catholic convention: S:t Hieronymus (St. Jerome), S:t Basilius (St. Basil), S:t Augustinus (St. Augustine), S:t Ambrosius (St. Ambrose), S:t Johannes Chrysostomos (Chrysostom), S:t Gregorius (Gregory), S:t Bernhard (Bernard), S:t Hilarius (Hilary), S:t Cyprianus (Cyprian), S:t Justinus (Justin), S:t Clemens (Clement), S:t Athanasius (Athanasius), S:t Epifanius (Epiphanius), S:t Ignatius (Ignatius), S:t Johannes av Damaskus (John of Damascus), S:t Bonaventura (Bonaventure), S:t Franciskus (Francis), S:t Dominikus (Dominic), S:t Thomas (Thomas), S:t Leo (Leo), S:t Paulus (Paul), S:t Efraim (Ephrem). Use Swedish quotation marks: primary quotation marks are "..." (English-style double quotes — these ARE the standard in Swedish typography), with '...' for nested quotes. Swedish Catholic theological terminology: Heliga Skriften (Holy Scripture), sakrament (sacrament), nåd (grace), synd (sin), frälsning (salvation), kyrkofader (Church Father), evangelium (Gospel), Första Moseboken (Genesis), Andra Moseboken (Exodus). Gospel references use Swedish Bible convention (e.g., "Joh 8:12" — note colon for verse separator per Swedish convention). The register should be formal scholarly Swedish suitable for theological commentary — formellt akademiskt språk, avoiding colloquial constructions. Proper use of Swedish diacritics (å, ä, ö) is critical. Note that Swedish uses "..." quotation marks (same visual style as English), unlike German/Polish/Dutch which use „...". Agents should NOT use „..." for Swedish. Psalm numbering: Swedish Bibles (Bibel 2000, Svenska Folkbibeln) use modern/Hebrew numbering — convert from Vulgate numbering where needed (e.g., Vulgate Psalm 89 → Psalm 90).
