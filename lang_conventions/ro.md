# Romanian

Use formal scholarly Romanian (stil academic/teologic). File suffix `_ro`. Latin alphabet, LTR (no `dir` attribute needed). `lang="ro"` on the `<html>` tag.

## Quotation marks

Use Romanian quotation marks „..." for standard quotations and «...» for nested quotes.

## Theological terminology

Sfânta Scriptură (Holy Scripture), sacrament (sacrament), har (grace), păcat (sin), mântuire (salvation), Părinte al Bisericii (Church Father), Evanghelie (Gospel), Facerea/Geneza (Genesis), Ieșirea/Exodul (Exodus).

## Bible references and psalm numbering

Gospel references use Romanian Bible convention (e.g., „Ioan 8,12" — note comma for verse separator per Romanian convention).

## Register

Formal scholarly Romanian suitable for theological commentary — stil academic, avoiding colloquial constructions. Proper use of Romanian diacritics (ă, â, î, ș, ț) is critical — agents must use the correct cedilla-below forms (ș, ț) not the cedilla forms (ş, ţ). Romanian theological tradition draws on both Latin Catholic and Byzantine Orthodox heritage; for this Catholic source material, prefer Greek-Catholic (Blaj) conventions where they differ from Orthodox usage.

## Agent pitfalls

- **English-style quotes**: Agents must be explicitly instructed to use Romanian quotation marks „..." (not English "...") as they default to English-style quotes.
- **Wrong diacritics**: Agents must use the correct cedilla-below forms (ș, ț) not the cedilla forms (ş, ţ).
- **Orthodox vs Greek-Catholic conventions**: For this Catholic source material, prefer Greek-Catholic (Blaj) conventions where they differ from Orthodox usage.
