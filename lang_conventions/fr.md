# French

Use formal scholarly/theological French (registre soutenu). File suffix `_fr`. Latin alphabet, LTR (no `dir` attribute needed).

## Biblical proper names

Follow French Bible tradition (Bible de Jérusalem / TOB): Moïse (Moses), Abraham (Abraham), Isaac (Isaac), Jacob (Jacob), Ésaü (Esau), Joseph (Joseph), Josué (Joshua), Isaïe (Isaiah), Jérémie (Jeremiah), Adam (Adam), Ève (Eve), Noé (Noah), David (David), Salomon (Solomon), Élie (Elijah), Élisée (Elisha), Aaron (Aaron), Gédéon (Gideon), Samson (Samson), Débora (Deborah), Ruth (Ruth), Samuel (Samuel), Ézéchias (Hezekiah), Jonas (Jonah), Daniel (Daniel).

## Saint names

Use saint/sainte (lowercase in running text per French convention):

saint Augustin (St. Augustine), saint Jérôme (St. Jerome), saint Basile (St. Basil), saint Ambroise (St. Ambrose), saint Jean Chrysostome (St. Chrysostom), saint Grégoire (St. Gregory), saint Bernard (St. Bernard), saint Hilaire (St. Hilary), saint Cyprien (St. Cyprian), saint Justin (St. Justin), saint Clément (St. Clement), saint Athanase (St. Athanasius), saint Épiphane (St. Epiphanius), saint Ignace (St. Ignatius), saint Jean Damascène (John of Damascus), saint Bonaventure (St. Bonaventure), saint François (St. Francis), saint Dominique (St. Dominic), saint Thomas (St. Thomas), saint Léon (St. Leo), saint Paul (St. Paul), saint Éphrem (St. Ephrem).

Note: "saint" is lowercase in French when used as a title before a name in running text (saint Augustin dit...). It is capitalized only when part of a proper noun (la fête de Saint-Augustin, l'église Saint-Ambroise).

## Quotation marks

Use French guillemets « ... » with non-breaking spaces inside (« texte ») for quotations. Nested quotes use double quotation marks "...". This is the standard French typographic convention — agents frequently omit the inner spaces or use English-style quotes.

## Theological terminology

Écriture sainte / Sainte Écriture (Holy Scripture), sacrement (sacrament), grâce (grace), péché (sin), salut (salvation), Père de l'Église (Church Father), Évangile (Gospel), Genèse (Genesis), Exode (Exodus), Pentateuque (Pentateuch), Vulgate (Vulgate).

## Bible references and psalm numbering

Gospel references use French Bible convention (e.g., « Jean 8, 12 » or « Jn 8, 12 » — note comma for verse separator). Psalm numbering follows the Vulgate convention used in the source text.

## Register

Formal scholarly French suitable for theological commentary — registre soutenu, avoiding colloquial constructions. Proper use of French diacritics is critical: é, è, ê, ë, à, â, î, ï, ô, û, ù, ç, œ.

## Agent pitfalls

- **English-style quotes**: Agents default to English "..." quotation marks instead of guillemets « ... » with inner spaces. Must be explicitly instructed.
- **Dropped accents on capitals**: Agents often drop accents on capital letters (É, À, etc.) which is incorrect in modern French typography.
- **Capitalized "Saint"**: Agents capitalize "saint" before names in running text. In French, it is lowercase (saint Augustin) unless part of a proper noun (Saint-Augustin).
