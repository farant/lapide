# German

Use gehobene Schriftsprache (formal scholarly German). File suffix `_de`. Latin alphabet, LTR (no `dir` attribute needed).

## Biblical proper names

Follow German Bible tradition (Einheitsübersetzung/Lutherbibel): Mose (Moses), Abraham, Isaak (Isaac), Jakob (Jacob), Aaron, David, Josua (Joshua), Jesaja (Isaiah), Jeremia (Jeremiah), Adam, Eva (Eve), Noach (Noah).

hl. Hieronymus (St. Jerome), hl. Basilius (St. Basil), hl. Augustinus (St. Augustine), hl. Ambrosius (St. Ambrose), hl. Chrysostomus (Chrysostom), hl. Gregor (Gregory), hl. Bernhard (Bernard), hl. Hilarius (Hilary), hl. Cyprian (Cyprian), hl. Justin (Justin), hl. Klemens (Clement), hl. Athanasius (Athanasius), hl. Epiphanius (Epiphanius), hl. Ignatius (Ignatius), Johannes von Damaskus (John of Damascus), hl. Bonaventura (Bonaventure), hl. Franziskus (Francis), hl. Dominikus (Dominic), hl. Thomas (Thomas), hl. Leo (Leo), hl. Paulus (Paul), hl. Ephräm (Ephrem).

## Quotation marks

Use German quotation marks „..." for standard quotations and ‚...' for nested quotes.

## Theological terminology

Heilige Schrift (Holy Scripture), Sakrament (sacrament), Gnade (grace), Sünde (sin), Erlösung/Heil (salvation), Kirchenvater (Church Father), Evangelium (Gospel), Genesis/Buch Genesis (Genesis), Exodus/Buch Exodus (Exodus).

## Bible references and psalm numbering

Gospel references use German Bible convention (e.g., „Johannes 8,12" — note comma for verse separator per German convention). Psalm numbering follows the Vulgate convention used in the source text.

## Register

Formal scholarly German suitable for theological commentary — gehobene Schriftsprache, avoiding colloquial constructions. Proper use of Umlaute (ä, ö, ü) and Eszett (ß) is critical.

## Agent pitfalls

- **English-style quotes**: Agents default to English quotation marks "..." instead of German „...". Must be explicitly instructed.
