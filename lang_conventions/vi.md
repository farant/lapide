# Vietnamese

Use formal scholarly Vietnamese (văn phong học thuật/văn chương). File suffix `_vi`. Vietnamese Latin alphabet (chữ Quốc ngữ), LTR (no `dir` attribute needed). `lang="vi"` on the `<html>` tag. Biblical proper names follow Vietnamese Catholic tradition (Nhóm Phiên Dịch Các Giờ Kinh Phụng Vụ): Môsê (Moses), Ápraham (Abraham), Ixaác (Isaac), Giacóp (Jacob), Aharon (Aaron), Đavít (David), Giôsuê (Joshua), Isaia (Isaiah), Giêrêmia (Jeremiah), Ađam (Adam), Evà (Eve), Nôê (Noah). Saint names use Thánh prefix following Vietnamese Catholic convention: Thánh Giêrônimô (St. Jerome), Thánh Basiliô (St. Basil), Thánh Augustinô (St. Augustine), Thánh Ambrôsiô (St. Ambrose), Thánh Gioan Kim Khẩu (Chrysostom), Thánh Grêgôriô (Gregory), Thánh Bênađô (Bernard), Thánh Hilariô (Hilary), Thánh Cyprianô (Cyprian), Thánh Giustinô (Justin), Thánh Clêmentê (Clement), Thánh Atanasiô (Athanasius), Thánh Êpiphaniô (Epiphanius), Thánh Inhaxiô (Ignatius), Thánh Gioan Đamascênô (John of Damascus), Thánh Bônaventura (Bonaventure), Thánh Phanxicô (Francis), Thánh Đa Minh (Dominic), Thánh Tôma (Thomas), Thánh Lêô (Leo), Thánh Phaolô (Paul), Thánh Êphrem (Ephrem). Use Vietnamese quotation marks: standard double quotation marks "..." for quotations and '...' for nested quotes. Vietnamese Catholic theological terminology: Kinh Thánh (Holy Scripture), bí tích (sacrament), ân sủng (grace), tội (sin), ơn cứu rỗi/cứu độ (salvation), Giáo phụ (Church Father), Phúc Âm (Gospel), Sáng Thế Ký (Genesis), Xuất Hành (Exodus). Gospel references use Vietnamese Bible convention (e.g., "Ga 8,12" for John 8:12 — note comma for verse separator per Vietnamese convention; Ga = Gioan, Mt = Mátthêu, Mc = Máccô, Lc = Luca). The register should be formal scholarly Vietnamese suitable for theological commentary — avoiding colloquial constructions and slang.

**Diacritics are absolutely critical.** Vietnamese uses 6 tone marks (sắc á, huyền à, hỏi ả, ngã ã, nặng ạ, ngang — unmarked) combined with special vowels (ă, â, ê, ô, ơ, ư) and the consonant đ. Every syllable must carry its correct tone mark and vowel form. Agents are highly prone to dropping tone marks or using incorrect tones, which changes word meaning entirely (e.g., bàn = table, bán = sell, bản = version). Explicitly instruct all agents to preserve full Vietnamese diacritics on every word. Double-check diacritics during review.

**Spacing**: Vietnamese is a monosyllabic language written with spaces between syllables. Multi-syllable words (often Sino-Vietnamese compounds) are written as separate syllables with spaces (e.g., "giáo phụ" not "giáophụ"). Agents must not merge syllables.

**Agent pitfalls**: (1) Dropping tone marks — the most common and most damaging error. (2) Using Northern vs Southern dialect inconsistently — use standard/Northern Vietnamese (Hà Nội standard) for formal writing. (3) Transliterating proper names inconsistently — always use the Vietnamese Catholic forms listed above, not ad hoc transliterations. (4) Leaving quotations untranslated in Latin or English — all quoted text must be translated into Vietnamese per project conventions.
