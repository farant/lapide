# Hindi

Use शुद्ध हिन्दी (formal/standard Hindi). File suffix `_hi`. Devanagari script, LTR (no `dir` attribute needed).

## Biblical proper names

Follow Hindi Bible tradition: मूसा (Moses), इब्राहीम (Abraham), याकूब (Jacob), इसहाक (Isaac), हारून (Aaron), दाऊद (David), यहोशू (Joshua), यशायाह (Isaiah), यिर्मयाह (Jeremiah).

## Saint names

Use संत (Sant) prefix. **Church Father names use Latinized forms** (not anglicized), preserving the Latin scholarly texture of Lapide's original.

Key mappings: संत अगस्टिनुस (Augustine), संत हिएरोनिमुस (Jerome), संत बासिलियुस (Basil), संत अम्ब्रोसियुस (Ambrose), संत क्रिसोस्तोमुस (Chrysostom), संत ग्रेगोरियुस (Gregory), संत बेर्नार्दुस (Bernard), संत हिलारियुस (Hilary), संत सिप्रियानुस (Cyprian), संत युस्तिनुस (Justin), संत क्लेमेन्स (Clement), संत अथानासियुस (Athanasius), संत एपिफ़ानियुस (Epiphanius), संत इग्नातियुस (Ignatius), दमिश्की योहन्नेस (John of Damascus), संत बोनावेन्तूरा (Bonaventure), संत फ़्रान्सिस्कुस (Francis), संत दोमिनिकुस (Dominic).

Names already close to Latin are kept as-is: थॉमस, लियो, पौलुस, एफ़्रेम.

## Quotation marks

Use standard double quotation marks "..." for quotations.

## Bible references and psalm numbering

Gospel references use standard Hindi Bible name यूहन्ना (e.g., "यूहन्ना 8:12").

## Register

Devanagari has no diacritics concern in the European sense, but proper use of matras (vowel signs), nukta (़), chandrabindu (ँ), and visarga (ः) is critical. Agents must produce proper Devanagari — watch for missing matras or incorrect conjuncts (संयुक्ताक्षर).

## Agent pitfalls

- **Anglicized Church Father names**: Agents default to anglicized forms like जेरोम, ऑगस्टीन instead of the required Latinized forms (हिएरोनिमुस, अगस्टिनुस). Must be explicitly instructed.
